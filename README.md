
# Linux Labs

Repositório para prática de Linux com foco em automação, scripts e desenvolvimento

## Objetivo
Desenvolver competências práticas em:
- Manipulação de arquivos e diretórios
- Automação com shell script
- Monitoramento de sistema
- Uso de ferramentas de linha de comando

## Estrutura do Projeto
linux-labs/
- scripts/
- logs/
- data/


## Scripts

### monitor.sh
Script responsável por coletar informações do sistema:
- Uso de CPU
- Uso de memória
- Registro em log

## Tecnologias
- Linux (Ubuntu)
- Bash
- Git
